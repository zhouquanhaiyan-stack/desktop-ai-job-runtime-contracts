# v0.1.0 Release Notes

## Summary

This is the first public release of **desktop-ai-job-runtime-contracts**, a lightweight, dependency-free Python library that defines the contract surface for running AI inference jobs in a local desktop environment.

## What's included

* **Job manifest schema** — typed JSON serialization/deserialization and validation helpers.
* **Runtime adapter boundary** — abstract `BaseRuntimeAdapter` plus a `FakeRuntimeAdapter` that demonstrates the full job lifecycle without external models or cloud services.
* **Execution trace model** — event-level recording with unique IDs, UTC timestamps, stage names, messages, and optional metadata.
* **Export guard stub** — a sample policy demonstration mapping artifact kinds to plan tiers.
* **Example manifests** — audio, image, and text job manifests for contract smoke testing.
* **Example runner** — a CLI script that loads a manifest, runs the fake adapter, and produces result artifacts and trace output.
* **Unit tests** — comprehensive test suite covering manifests, adapters, traces, export guard, and example execution.
* **GitHub Actions CI** — syntax checks, unit tests, and example runner verification across Python 3.10, 3.11, and 3.12.
* **Documentation** — architecture overview, job manifest reference, adapter contract guide, execution trace docs, example walkthrough, and export guard description.

The entire package is **standard-library-only** with zero external dependencies.

## How to verify locally

```bash
# Verify syntax
python -m py_compile src/desktop_ai_job_runtime_contracts/*.py examples/run_fake_job.py

# Run the full test suite
python -m unittest discover -s tests

# Run the default audio example
python examples/run_fake_job.py

# Run the image example
python examples/run_fake_job.py examples/fake_image_job.json

# Run the text example
python examples/run_fake_job.py examples/fake_text_job.json
```

## Compatibility

* Requires Python 3.10 or later.
* No external dependencies — pure Python standard library only.
* Compatible with Windows, macOS, and Linux.

## Security and privacy notes

* The `FakeRuntimeAdapter` performs **no real AI inference**. It creates small text files as placeholder artifacts for contract testing.
* This package contains **no real credentials, private paths, or production service details**.
* There is **no payment, licensing, or email logic** in this release.
* No network calls are made during execution.

## Known limitations

* The `ExportGuard` is a demonstration stub only — it is not a real authorization or licensing system.
* Only `FakeRuntimeAdapter` is included; production adapters (ONNX, llama.cpp, etc.) are planned but not yet implemented.
* Streaming trace support for long-running jobs is not yet available.
* Artifact hashing and integrity verification are not yet implemented.
